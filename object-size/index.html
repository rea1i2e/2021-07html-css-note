レスポンシブを考慮したサイズ指定

html-css-note

  • 1.px指定

    295px

    width: 295px;
    height: 295px;
    指定は簡単だが、画面幅が変化してもサイズが変化しない

  • 2.%指定

    25%

    width: 25%;
    height:295px;
    親要素に対する割合で指定。
    幅の指定は比較的簡単だが、高さの指定は困難。
    正方形を作ることもできない。
    ここでは高さはpx指定とした。

  • 3.vw指定

    90vw x 割合

    width: calc((100vw - 50px ) / 4 );
    height: calc((100vw - 50px ) / 4 );
    画面幅から余白分を引いた値に割合をかけている。
    高さに対しても、画面幅の単位（vw）を使えるので、正方形を作るなどアスペクト比を保つことができる。

  • 4.widthはvw指定、max-widthはpx指定

    90vw x 割合

    width: calc((100vw - 50px ) / 4 );
    max-width: 295px;
    height: calc((100vw - 50px ) / 4 );
    max-height: 295px;
    「3.」の指定に加えて、最大値を設定。
    途中までは画面幅に応じてサイズを大きくする。最大値になるとそれ以上大きくならない。

VScodeの便利な使い方

  • HTML・CSS noteに戻る

emmetを使うと、入力が楽

  • !+returnで、htmlのテンプレートを入力
  • ul>li*3で、ul直下にliを3つ入力。tabで次のli位置へ移動できる
  • script:src　→　<script src=""></script>

VScodeの便利なショートカットキー

行を下に移動 Option + ↓
行を下にコピー Option + Shift + ↓
下に行を挿入 Command + Retern
上に行を挿入 Command + Shift + Retern
任意の位置にカーソル追加（マルチカーソル） Option + クリック
真下にカーソル追加（マルチカーソル） Option + ↓
単語の選択 Command + d
選択中の単語と同じ単語を1つ追加で選択 単語選択中に、Command + d
選択中の単語と同じ単語をすべて選択 単語選択中に、Command + l

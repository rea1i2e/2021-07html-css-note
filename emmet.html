Emmetの便利な使い方

  • HTML・CSS noteに戻る

HTMLの記法

  • Emmetの公式サイトにチートシートがある
  • 要素名を入力するだけで、例えば「li」 →　「<li></li> 」と展開される
  • aタグのように、よく使う属性まで展開される場合もある。「｜」はカーソルを示し、複数ある場合tabで移動できる
    • ＜br＞ 「a」　→　「<a href="|">|</a>」
  • 「要素名」＋「.」＋「クラス名」とすると、クラス名をつけた要素が展開される。
    例えばこんな感じ　「div.container」　→　「<div class="container"></div> 」
    IDも同様にできる
  • IDと複数のクラスを同時に指定することも可能
    例えばこんな感じ　「dev#reset.btn.disabled」　→　「<dev id="reset" class="bnt disabled"></dev> 」
  • その他の属性は、要素[属性名＝"属性値]と入力すると展開してくれる
  • 属性値に連番をつけたい場合は、「$」を使う
    例えばこんな感じ「div.item*$3」　→
              <div class="item1"></div>
          <div class="item2"></div>
          <div class="item3"></div>
    
          「$」を増やすと、連番の桁数が増える
  • 「main>section*3+div.btn」のように入力すると、複数階層に複数の要素を展開できる
              <main>
            <section></section>
            <section></section>
            <section></section>
            <div class="btn"></div>
          </main>
  • 「^（キャレット）を使うと、上の階層に戻って展開することができる」
    例えばこんな感じ　main>section*2^footer 
                <main>
            <section></section>
            <section></section>
          </main>
          <footer></footer>
  • クラス名やID名のみ入力すると、div要素が展開される
  • ulやtable要素の下にクラス名を入力すると、liやtrなどの子要素が暗黙で展開される
  • 「lorem」　→　ダミーテキストが展開される
    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ducimus temporibus voluptates quo officia sequi unde est magnam minus corrupti! Quidem culpa cum modi accusamus eaque maiores quo similique expedita consequuntur.
  • コマンドパレット　→　対象の範囲を選択して、command+shift+p
    選択範囲をタグで囲う　→　wrapと入力して、wrap with Abbreviationを選択後、タグを入力
    複数行の選択範囲を行ごとにタグで囲う　→　wrapと入力して、Wrap Individual Lines with Abbreviationを選択
    ※なぜかこれはうまく動作しなかった
  • imgタグ上で、コマンドパレットを呼び出し、updateと入力してUpdate Image Sizeを選択すると、imgタグに自動でサイズが展開する
  • imgのサイズの属性に320*2などと入力して、コマンドパレットを呼び出し、evaluateと入力すると、計算した値を返してくれる
    ※コマンドは出てきたけど、選択しても実行されなかった。

CSSの記法

CSSではプロパティに対してEmmetの記法が変える

  • 「m」　→　「margin: ;」
  • 「m10」　→　「margin: 10px;」
  • 「m10p」　→　「margin: 10%;」
  • 「m10r」　→　「margin: 10rem;」
  • 「m0-auto」　→　「margin: 0 auto;」
  • 「mb10」　→　「margin-bottom: 10px;」

floatを含む親要素に下記のようなclearfixクラスを指定することでfloatを解除できる。

.clearfix:after {
  content: "";
  display: block;
  clear: both;
}

floatを指定したオブジェクトの親要素の高さ

floatを指定したオブジェクト(.box1と.box2)の親要素(.box-wrapper)は、高さを指定しない限り高さがゼロになる
